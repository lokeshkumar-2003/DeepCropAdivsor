# DeepCropAdvisor: Region-Aware Crop Disease Detection and Treatment Recommendation

## Overview
DeepCropAdvisor is an AI-powered agricultural decision-support system designed to assist farmers in early crop disease diagnosis and effective treatment planning. The system combines deep learning–based image classification with intelligent advisory recommendations to provide both disease detection and actionable treatment guidance.

The proposed solution uses an EfficientNetB3-based Convolutional Neural Network (CNN) to analyze crop leaf images and accurately identify plant diseases along with infection severity levels. Unlike traditional agricultural disease detection systems that only classify diseases, DeepCropAdvisor integrates a Large Language Model (LLM)-based advisory module capable of generating region-aware treatment recommendations.

Based on the detected disease and the farmer’s region, the system suggests:
- Appropriate chemical pesticides
- Organic and eco-friendly treatment alternatives
- Recommended dosage and usage instructions
- Preventive measures to reduce future infections

By combining computer vision with AI-driven expert guidance, DeepCropAdvisor helps bridge the gap between crop disease identification and real-world agricultural treatment support. The system aims to improve crop productivity, support sustainable farming practices, and enable farmers to make informed decisions efficiently.

---

## Features
- Crop disease detection using leaf images
- Infection severity analysis
- EfficientNetB3-based deep learning model
- Region-specific treatment recommendations
- Chemical and organic solution suggestions
- Dosage and preventive guidance
- AI-powered LLM advisory support

---

## Technologies Used
- Python
- TensorFlow / Keras
- EfficientNetB3
- OpenCV
- Large Language Models (LLMs)
- Flask / FastAPI
- React.js (Optional Frontend)

---

## Objective
The main objective of this project is to provide farmers with an intelligent, accessible, and practical agricultural assistant that not only detects crop diseases accurately but also delivers actionable treatment recommendations tailored to regional farming conditions.

---

## Future Enhancements
- Multi-language farmer support
- Real-time mobile application integration
- Weather-aware disease prediction
- Offline advisory support for rural areas
- IoT-based smart farming integration

---

## Conclusion
DeepCropAdvisor combines artificial intelligence, deep learning, and natural language processing to create a smart agricultural assistance platform. The system not only identifies crop diseases with high accuracy but also provides actionable treatment recommendations, helping farmers improve productivity and adopt sustainable farming practices.
